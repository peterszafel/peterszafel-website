# Peter Szafel Photography Website

This is a complete static website. You can upload the whole `photography-website` folder to your web server.

## What is included

- Homepage with purple, black, and gold styling
- About Me section with Peter's biography
- Portfolio gallery using Peter's real photos
- Portfolio filters for Portraits, Weddings, Landscapes, and Creative work
- Contact section with phone number
- Links section with Facebook and phone link
- Upload-ready zip file

## What to edit before going live

1. Open `index.html` in a text editor.
2. The email address has been set to `peterszafel@gmail.com`.
3. Replace the Instagram and TikTok links if you want to use them.
4. If you want a different hero photo, replace `assets/images/hero.jpg` with your chosen photo using the same filename.

## How to view it

Double-click `index.html` to open it in your browser.

## How to upload it

Upload everything inside the `photography-website` folder to your hosting server. Make sure `index.html` is at the public root of your website, usually called `public_html`, `www`, or `htdocs`.

If your host lets you upload a zip file, upload `photography-website.zip` and extract it on the server.
